U-Cursedn't

Versión 26.6.1 Junio 2026 Major
Soporte oficial para Mozilla Firefox MV3, refactorización global de arquitectura (DRY), Fuzzy Search, Infinite Scroll y pipeline de empaquetado seguro.
+
Nuevas Características
  • Compatibilidad con Mozilla Firefox: Adaptación completa del manifiesto a las especificaciones de Gecko. Reemplazo de Service Workers por Event Pages (`background.scripts`), re-estructuración de la barra lateral mediante `sidebar_action` y declaración explícita de `data_collection_permissions` para auditorías de privacidad.
  • Búsqueda Difusa en Navegación (Fuzzy Search): Incorporación de la utilidad global `fuzzySearch.js` y `fuzzySearchHelper.js` para filtrar dinámicamente cualquier menú o listado (historial, foros, ramos) mediante coincidencia secuencial de caracteres de manera no-estricta.
  • Autocarga Asíncrona (Infinite Scroll): Añadido `autoPreload.js` acoplado a un `IntersectionObserver` que carga las siguientes páginas de hilos en foros o registros en el historial de manera fluida a demanda de scroll. Cuenta con codificación inteligente automática para caracteres latinos (tildes, eñes) y un filtro anti-duplicación de DOM por ID y Anchor.
  • Módulo de Silencio "Tour Guide": Añadido switch maestro en la configuración global para deshabilitar las alertas informativas recurrentes de descubrimiento de funciones.
↑
Mejoras y Optimizaciones
  • Persistencia de Calendario y Multi-vista: El `weekCounter.js` fue reescrito para leer la fecha directamente de los parámetros de URL, haciéndolo compatible con los modos Mes, Semana, Día y Agenda. Ahora escanea todas las celdas del mini-mapa d3 para no fallar si el usuario solo tiene clases en días específicos (o sábados), guardando el estado en un caché semestral indexado para un renderizado inmediato.
  • Estandarización de Promesas de Almacenamiento: Removidos los viejos e inestables callbacks asíncronos vinculados a `chrome.runtime.lastError`. Toda la extensión opera ahora bajo promesas nativas de la API `browser.*` a través de la inyección de `browser-polyfill.min.js`.
🔧
Mejoras Técnicas y de Arquitectura
  • Consolidación DRY de Código: Eliminado por completo el script redundante `extensionUtils.js`. Se creó la librería centralizada `utilities.js` organizada en sub-namespaces (`Storage`, `DOM`, `Browser`, `Ucursos`, `Date`) inyectada globalmente al inicio de los manifiestos, incluyendo cláusulas de escape automáticas para evitar excepciones por invalidación de contexto.
  • Pipeline Avanzada de Build y Linters: El script `build.py` ahora integra un sistema de análisis estático que detiene la compilación si detecta JSONs mal formados o comas consecutivas en JS. Excluye automáticamente archivos basura de desarrollo (`*.old`, caches de python) y gestiona un archivo rotativo temporal (`/archive`) limitado de manera estricta a las últimas 3 versiones empaquetadas.
Versión 26.3.4 Marzo 2026 Hotfix
Correcciones y optimizaciones en notificaciones de tareas y etiquetas de semana
✓
Correcciones
  • Contexto de Posicionamiento de Notificaciones: Asegurada la presencia de position: relative en el elemento padre del menú de tareas para proporcionar contexto correcto de posicionamiento absoluto. Previene problemas de desplazamiento de elementos UI cuando el padre carece de contexto de posicionamiento.

Versión 26.3.2 Marzo 2026 Major
Panel Lateral de IA, conversaciones multi-turn, efecto Dock de macOS y publicación automatizada en Chrome Web Store
+
Nuevas Características
  • Panel Lateral de IA (Side Panel): Nuevo panel lateral nativo de Chrome con el asistente UCursedn't AI directamente integrado en el navegador. Incluye historial multi-chat persistente mediante chrome.storage (arquitectura amnesia-proof MV3), detección automática de Gemini Nano para procesamiento local, y botón de acceso rápido desde el popup de la extensión.
  • Nuevo esquema de versiones: U-Cursedn't adopta un sistema de versiones basado en fecha (AA.MM.S): año (dos dígitos), mes y número de semana dentro del mes. La versión 26.3.2 corresponde a la primera semana de marzo de 2026.
↑
Mejoras y Optimizaciones
  • Conversaciones Multi-turn en Gemini: El historial completo de la conversación se envía ahora al modelo en cada consulta, con conversión correcta de roles (assistant → model). Modelo actualizado a Gemini Flash Lite Latest.
  • Animaciones — Efecto Dock de macOS: navigationAnimations.js completamente refactorizado para usar una transformación de escala spring-eased (cubic-bezier) más ligera y fluida. Las secciones colapsables también aplican el efecto de magnificación dock a sus elementos internos al pasar el cursor.
  • Sonido de Entrega de Tareas: Reemplazado el sintetizador Web Audio API por reproducción de audio mediante iframe de YouTube. Ahora se activa en todas las páginas de tareas (tareas/*), no solo en la vista de detalle. El iframe se elimina automáticamente tras 15 segundos para evitar fugas de memoria en el DOM.
  • Interacción con Foros: Patrones de URL expandidos para cubrir los foros institucionales de Ingeniería y U-Chile (foro_institucion, uchile/*/0/*/1/foro/). Refactor interno para reducir duplicación y mejorar la delegación de eventos.
🔧
Mejoras Técnicas
  • Actualización del Manifiesto: Añadido permiso sidePanel y entrada side_panel.default_path. Política de seguridad de contenido (CSP) configurada para permitir conexiones a la API de Gemini. Correcciones en content scripts: popup.js restringido a U-Cursos, navigationAnimations.js movido a document_idle.
  • Íconos estandarizados: Migración de clases fa-solid a fa-regular en todos los módulos (achievementsGen, easyCopy*, menuGen, muchoTexto, renameCourses, weekCounter) para mayor coherencia visual.
  • Publicación Automatizada en Chrome Web Store: Nuevo GitHub Action que publica automáticamente la extensión al hacer push a main, usando furkanipek/chrome-extension-action.
